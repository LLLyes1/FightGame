# M4 新角色定位说明

## 选择结果

M4 第一名新增角色定为：`bastion`

## 选择原因

当前角色结构是：

- `vanguard`：标准型
- `sky_drifter`：高机动空中特化型

缺的最明显空位是“重型稳定角色”。

`bastion` 的加入主要为了验证三件事：

1. 项目能否稳定承载更重、更慢的角色
2. 击飞、回场、压边差异是否仍然可读
3. 新角色是否能在两把旧武器和一把新武器下保持身份一致

## 角色身份

`bastion` 是重型 anchor。

他的特点不是抢先手速度，而是：

- 更强的站桩稳定性
- 更高的受击容忍
- 更慢但更扎实的推进

## 如何移动

- 地面更慢，但刹车更稳
- 空中位移更短，空中修正更弱
- 跳跃高度和横移都更保守
- dash 有明确前摇和更重的恢复感

## 如何回场

- 可以回，但不轻松
- 依赖更早决策和更短路径
- 不像 `sky_drifter` 那样能在空中二次修正很多次

## 如何赢

1. 通过重量和稳定性扛住交换
2. 把对手逼到边缘后拿更高收益的击飞

## 如何输

1. 被高机动角色持续拉扯
2. 回场路线被读后更难补救

## 和现有角色的区分

- 相比 `vanguard`：更慢、更重、更吃路线预判
- 相比 `sky_drifter`：更不灵活，但更能承受击飞与换血
