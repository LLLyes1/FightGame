# M4 新武器定位说明

## 选择结果

M4 第一把新增武器定为：`halberd`

## 选择原因

当前项目已有两把武器：

- `saber`：短距离、快节奏、追击压制
- `hand_cannon`：长距离、投射物牵制、斜线压制

所以第三把武器不再追求“更快”或“更远”，而是补一个中距离控场位。

`halberd` 的任务是：

1. 用长判定建立前方 lane control
2. 用更高承诺的收招换取更稳定的空间优势
3. 让玩家明显感觉到“这不是 rushdown，也不是 zoning”

## 战斗语言

`halberd` 的核心语言是：

- 先占位
- 再刺出
- 用长柄范围逼对方绕路或抢跳

它不是连续贴脸压制武器，也不是安全远射武器。

## 最强的 2 个场景

1. 中距离地面拦截
2. 对上跳和回场路线的提前封锁

## 最怕的 2 个场景

1. 被贴脸后进入高频近身换招
2. 连续空中追击时需要快速改向与多段纠错

## 关键槽位设计方向

- `light_ground`: 快速前刺，建立基础 poke
- `heavy_ground`: 高承诺横斩，承担终结与击飞
- `up_ground` / `up_air`: 明确承担反空职责
- `down_ground`: 低位扫线，限制贴地接近
- `special_side`: 强化中距离推进压迫
- `special_up`: 强调垂直控制，而不是强回场作弊

## 和现有武器的区分

- 相比 `saber`：更长、更慢、更讲站位
- 相比 `hand_cannon`：更需要亲自占线，不靠投射物拖节奏
