# M5 阵容观察矩阵

## 1. 用途

这份矩阵不是“最终平衡答案”，而是 `3x3` 阵容当前的统一观察底稿。

字段解释：

- `机动感`：地面与空中整体修正空间
- `对空`：应对跳入和上方交互的能力
- `压边`：把对手逼向边线后的持续压迫能力
- `回场容错`：掉台后回来的稳定度
- `命中收益`：单次确认后的伤害/击飞价值

## 3. 当前最值得盯的组合

- 最稳通用组：`vanguard + saber`
- 最高压边风险组：`bastion + halberd`
- 最高回场风险组：`bastion + hand_cannon`
- 最高拉扯安全性观察组：`sky_drifter + hand_cannon`

## 4. 当前最缺的 archetype 空位

如果下一轮继续扩展，当前阵容最缺的不是“再来一个中距控线”，而是下面二选一：

- 高风险爆发武器：让命中收益更极端，但容错明显下降。
- 高速贴身爆发角色：机动更强，但承伤与回场更差。

原因很简单：当前三把武器已经覆盖了近压、远牵制和中距控线，下一步更适合补“极端节奏”而不是补“中间值”。
