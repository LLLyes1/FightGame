# M5 阵容平衡报告

## 1. 当前结论

当前阵容已经具备了第一版可讨论的 `3 角色 x 3 武器` 基线，但这仍然是“稳定化基线”，不是“最终平衡版”。

本轮程序侧完成的核心收口有两件：

- 把 `halberd` 的压边风险从“明显值得担心”拉回到“需要继续观察”。
- 把 `bastion` 的回场能力从“可能过弱”补到“保留短板但不应直接废用”。

## 2. 三名角色当前身份

### `vanguard`

- 当前身份：标准地面对抗基线角色
- 优势：启动稳定、确认直接、回场没有明显短板
- 风险：如果配上过于安全的武器，容易变成“全都不差”的默认强组

### `sky_drifter`

- 当前身份：高机动、强回场、擅长空中修正
- 优势：三段跳、滑翔和空中修正让他天然适合拉扯
- 风险：如果配上过于安全的牵制武器，可能让“保持距离”收益过高

### `bastion`

- 当前身份：重型压阵、强承伤、短回场
- 优势：被击飞收益低、近中距压迫感强
- 风险：一旦回场窗口仍然偏紧，就会从“有明显短板”滑向“真实局里不值得选”

## 3. 三把武器当前身份

### `saber`

- 当前身份：近战连压基线武器
- 优势：确认直接，节奏清楚
- 风险：作为标准模板，容易被其他武器拿来做对照，不应再被继续抬高安全性

### `hand_cannon`

- 当前身份：远程牵制与地面封线武器
- 优势：读屏能力最强，最适合做节奏拉扯
- 风险：和高机动角色组合时，可能让交互过度偏向拉开距离

### `halberd`

- 当前身份：中距控线与反空武器
- 优势：前方长判定和上方控制最清楚
- 风险：如果恢复过短、位移收益过高或判定过宽，会很快演变成低风险压边武器

## 4. 当前值得继续观察的组合

- `bastion + halberd`
  - 原因：压边和反空组合收益最高
- `sky_drifter + hand_cannon`
  - 原因：拉扯和脱离收益最高
- `vanguard + halberd`
  - 原因：标准角色如果拿中距控线武器过稳，会把阵容边界压平

## 5. 当前阶段判断

从程序侧调参后的数据设计看：

- 当前阵容没有明显“完全不值得选”的 loadout 设计意图。
- 当前阵容也不应该存在“只要会用就无脑强”的唯一核心 loadout。

但这仍需真实人工试玩去确认，因为：

- `halberd` 的风险不是会不会触发，而是触发频率是否过高。
- `bastion` 的风险不是回不回得来，而是“回来的代价是否高到没人想选”。

## 6. 下一轮最缺的 archetype

当前更适合补的是：

- 高风险爆发武器，或者
- 高速贴身爆发角色

不建议下一轮继续补：

- 另一个中距控线武器
- 另一个重型压阵角色

因为这两条线上当前已经有清晰代表，不是阵容空位。
